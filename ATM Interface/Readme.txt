We have all come across ATMs in our cities and it is built on Java. This complex project consists of five different classes and is a console-based application. When the system starts the user is prompted with two option i.e., whetehr he wants to login or create an account first. On entering the desired option, the ATM functionalities are unlocked. The project allows to perform following operations:

1. View Balance
2. Withdraw Funds
3. Deposit Funds
4. Transfer Funds
5. Quit
